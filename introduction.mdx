## Key Features

<CardGroup cols={2}>
  <Card title="AI-Powered Conversations" icon="brain">
    Powered by Groq's LLaMA 3.3 70B model for natural, contextual responses with customizable personality prompts
  </Card>
  <Card title="Intelligent Spam Protection" icon="shield">
    Automatic detection and prevention of rapid messaging and repetitive content with configurable thresholds
  </Card>
  <Card title="Automated Messaging" icon="clock">
    Periodic message posting system with smart rotation to prevent repetition
  </Card>
  <Card title="Command Interpretation" icon="wand-magic-sparkles">
    AI-powered command understanding - mention the bot with natural language to execute commands
  </Card>
  <Card title="Member Events" icon="users">
    Customizable welcome and farewell messages with embedded cards
  </Card>
  <Card title="Admin Controls" icon="user-shield">
    Message purging, spam system management, and permission verification tools
  </Card>
</CardGroup>

## Built With Modern Technology

ChimBot leverages cutting-edge technologies:

- **discord.py** - Powerful Python library for Discord API
- **Groq API** - Ultra-fast AI inference with LLaMA 3.3 70B
- **Python 3.x** - Async/await for high-performance bot operations
